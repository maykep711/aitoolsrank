# AIToolsRank

> Programmatic AI tools & SaaS reviews publisher — React + Vite + TypeScript, SEO-first architecture, affiliate monetization.

## Stack

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + TypeScript |
| Styling | Tailwind CSS (dark mode first) |
| Routing | React Router v6 (SPA) |
| SEO | react-helmet-async + Schema.org JSON-LD |
| Data fetching | TanStack Query v5 |
| State | Zustand |
| Validation | Zod |
| Analytics | GA4 + GTM + Microsoft Clarity |
| Indexing | IndexNow |

---

## Architecture

```
src/
├── types/              # TypeScript interfaces (Tool, Cluster, SEOMeta, etc.)
├── data/
│   ├── tools.ts        # All tool data + query helpers
│   └── clusters.ts     # Cluster metadata (5 initial clusters)
├── lib/
│   ├── seo.ts          # Title/desc/schema builders + internal linking engine
│   └── tracking.ts     # GA4 + GTM + Clarity + IndexNow + SPA tracking
├── components/
│   ├── Layout/
│   │   ├── MainLayout.tsx
│   │   ├── Header.tsx
│   │   └── Footer.tsx
│   └── Review/
│       ├── ToolCard.tsx       # Reusable tool card with affiliate CTA
│       └── PricingTable.tsx   # Responsive pricing plan table
└── pages/
    ├── HomePage.tsx
    ├── ClusterPage.tsx         # /:cluster — category landing pages
    ├── ToolReviewPage.tsx      # /tools/:slug
    ├── ToolAlternativesPage.tsx # /tools/:slug/alternatives
    ├── ToolPricingPage.tsx     # /tools/:slug/pricing
    ├── ToolComparisonPage.tsx  # /tools/:slug/comparison
    ├── ToolBestForPage.tsx     # /tools/:slug/best-for
    ├── ToolIntegrationsPage.tsx # /tools/:slug/integrations
    ├── ToolUseCasesPage.tsx    # /tools/:slug/use-cases
    ├── CompareVsPage.tsx       # /compare/:tool1-vs-:tool2
    └── NotFoundPage.tsx
```

## URL Structure (7 page types per tool)

```
/                                    → Homepage
/:cluster                            → Cluster landing (e.g. /ai-writing-tools)
/tools/:slug                         → Review page
/tools/:slug/alternatives            → Alternatives
/tools/:slug/pricing                 → Pricing breakdown
/tools/:slug/comparison              → vs Competitors
/tools/:slug/best-for                → Best For / Use Cases
/tools/:slug/integrations            → Integrations list
/tools/:slug/use-cases               → Detailed use cases
/compare/:tool1-vs-:tool2            → Head-to-head comparison
```

## Getting Started

```bash
# Clone
git clone https://github.com/maykep711/aitoolsrank.git
cd aitoolsrank

# Install
npm install

# Configure environment
cp .env.example .env
# Fill in VITE_GA4_ID, VITE_GTM_ID, VITE_CLARITY_ID

# Dev server
npm run dev

# Production build
npm run build
```

## Adding a New Tool

1. Add a new `Tool` object to `src/data/tools.ts`
2. Assign it to the appropriate `clusters` array
3. All 7 page types are generated automatically from the data
4. Internal links are auto-generated by `src/lib/seo.ts`

---

## Monetization

- Affiliate links via `tool.affiliateUrl` field
- Click tracking via `trackAffiliateClick()` on every CTA
- All affiliate links have `rel="noopener noreferrer nofollow"`
- Sponsored placement supported via `tool.sponsored: true`

---

## SEO Features

- Unique title/description per page type via template system
- Schema.org JSON-LD: SoftwareApplication, Review, FAQPage, BreadcrumbList, ItemList
- Canonical URLs for all pages
- SPA route tracking for accurate GA4 data
- Internal linking engine connects related pages automatically
- RSS feed at `/feed.xml`
- IndexNow integration for instant indexing
